Ejercicio 7: Aplicación de gestion de biblioteca

La aplicación simula una aplicación de gestion de biblioteca que usuaria un bibliotecario que quiere mantener el registro de los libros y prestamos de una biblioteca

Hay 4 tablas, Libro, Autor, Prestamo y usuario

Para insertar datos ahora mismo solo se puede a traves del boton Reiniciar Base de Datos, que reinicia y añade los datos que se encuentran en los ficheros csv, para añadir datos nuevos habría que modificar estos ficheros

Pueden hacerse prestamos de libros, para ellos se ha de seleccionar un libro y un usuario, el libro seleccionado no puede estar ya prestado, y tanto el libro como el usuario introducido tiene que existir (se introducen por id) esto añadira una nueva entrada en la base de datos Pretamo en la que figurará el libro y el usuario que lo ha tomado prestado

Se puede devolver un libro, para ello simplemente se selecciona el id del libro a devolver

El botón mostrar libros y prestamos muestra una lista de todos los libros en la BD, asi como los usuarios que los han tomado prestados

Se puede hacer búsqueda de libros, buscando por nombre autor o genero, la busqueda se puede realizar tanto por texto completo como parcial, no distingue mayusculas y minusculas

Un Libro solo puede ser prestado por un usuario a la vez, para simplificar supondremos que solo tenemos un ejemplar de cada libro, un usuario puede tomar prestados tantos libros como quiera
